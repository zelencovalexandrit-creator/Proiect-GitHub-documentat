# Sait Companie de Construcție

## Descriere proiect
Acesta este un site simplu pentru o companie de construcții, realizat pentru prezentarea serviciilor, proiectelor și contactarea companiei de către clienți. Site-ul este creat folosind HTML și CSS, cu un design responsive pentru a fi accesibil pe diferite dispozitive.  

## Tehnologii folosite
- **HTML5** – structurarea conținutului site-ului
- **CSS3** – stilizarea paginilor și design responsive
- **JavaScript (opțional)** – pentru elemente interactive (dacă ai folosit)
- **Imagini și iconițe** – pentru prezentarea vizuală a serviciilor
